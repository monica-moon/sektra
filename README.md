# SEKTRA — The Fauna Collection

A single-page launch announcement for SEKTRA, a luxury house of animal-formed perfumes.
Bold editorial design, built as a static site with no framework or build step.

## Stack
- Static HTML / CSS / vanilla JS
- Google Fonts: Cormorant Garamond (display), Archivo (body), Montserrat (labels)
- Transparent PNG flacon cutouts and a looping hero video

## Structure
- `index.html` — markup and inline JS data
- `styles.css` — full stylesheet
- `script.js` — nav, scroll reveals, product modal, story modal, scent-finder quiz
- `assets/` — flacon cutouts, hero video, atelier photograph

## Run
Open `index.html` in a browser, or serve the folder:

```
python3 -m http.server
```
